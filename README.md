## Description

The API project for all VPI Triangle Fraternity applications

## Installation

```bash
$ npm install
```

## Running the app locally

```bash
# development
$ npm run start && func host start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
